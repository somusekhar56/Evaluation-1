# 2 What is a Merge Conflict? How to fix it?

A merge conflict happens when Git cannot automatically combine changes between two branches.
* This usually occurs when:
   Two people edit the same line in a file.
  * Manually fix the file:
    To fix - open the file mentioned the message
    
             Edit and remove the conflict markers(<<<< HEAD,==== Main branch, >>>> feature1)
    
             Save, then run.
    
             Marks the changes resloved.
    
# 3 what is git rebase?
git rebase is a Git command used to move or reapply commits from one branch on top of another branch.
